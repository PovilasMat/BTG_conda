This doc contains software needed on the laptops for the course, which can not be included in a conda-environment. List your wishes here, with a brief description, and links to installation


##Bandage

Software to visualize genome assembly graphs, as generated by some assemblers (including SPAdes). Exists as pre-compiled binaries:

https://rrwick.github.io/Bandage/


##ENA webin file uploader

https://ena-docs.readthedocs.io/en/latest/submit/fileprep/upload.html#using-webin-file-uploader

##ENA ftp downloader

https://github.com/enasequence/ena-ftp-downloader/
